Keep demos in sync with your product.

AutoDemo generates interactive walkthroughs and videos from a running web app. It’s designed for builders, teams, and tool-use agents.

Open interactive demo Watch video
Stagehand-first
Use LLM-native act steps, with Playwright fallbacks for precision.
Bun-first CLI
Fast iteration, Ink TUI for humans, and deterministic modes for CI.
Agent-ready
MCP server over stdio so tools/agents can list/generate/run demos.

Outputs

  • index.html (interactive walkthrough)
  • run.json (timings + step metadata)
  • steps/*.png (one screenshot per step)
  • video.mp4 (optional) with cursor + click rings

Try it

bunx autodemo init
bunx autodemo run signup --url http://localhost:3000
bunx autodemo run --interactive
bunx autodemo mcp --no-tui

Tip: add stable selectors like data-testid for the best recordings.
